3. Operational Modes

Force Mode

The die will inevitably land on the number corresponding to the zone you tapped. Essential for standard forcing techniques.

Avoid Mode

The die is programmed to avoid landing on the tapped zone's number. The result will be a random selection from the remaining five faces.

Pro Tip: Once the target number you want to avoid has been tapped, you no longer need to tap the same area. You can freely tap anywhere on the screen for the subsequent rolls!

⚠️

Warning

Although this mode is designed for Russian Roulette routines, it is highly discouraged to perform Russian Roulette using real dangerous objects, as the app is not yet 100% reliable. Please perform safely.

Sequence Mode

Finger positioning is ignored. The die follows a predetermined numerical pattern established in the settings. Once exhausted, the simulator reverts to random behavior.

4. Performance Tips

  • Normal Start Configure the slider to allow a set number of initial, genuinely random rolls. This subtly disarms spectator suspicion before applying any force.
  • Practice Grid Enable the visual grid overlay during rehearsals to build muscle memory for the six zones. Ensure it is disabled prior to performing.
